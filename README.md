**TeamLegacy Studio** is a powerful, browser-based Non-Linear Video Editor (NLE). It combines the convenience of a web application with the heavy-duty power of a desktop editor. No installation required, no privacy concerns—your files stay on your machine.

## 🚀 Core Features
*   📁 **Zero Installation:** Runs entirely in your web browser via `teamlegacy-editor-pro.html`.
*   🔒 **100% Private:** Everything is processed locally; no files are ever uploaded to a server.
*   🎞️ **Multi-Track Timeline:** Advanced layering with support for multiple video (V) and audio (A) tracks.
*   🖱️ **Drag & Drop Workflow:** Seamlessly move clips along the timeline and manage your Project Bin.
*   ✂️ **Non-Destructive Editing:** Trim, split, and re-scale clips without ever losing your original data.
*   ⚡ **Fast Export:** Render your final creation directly to high-quality `.WebM` format.

## 💎 What’s New in v2.0 (The Massive Upgrade)
The v2.0 update introduces a professional-grade feature set, significantly expanding the engine's capabilities:

### 🎞️ Advanced Timeline Engine
*   **Expanded Tracks:** Now supports up to **5 tracks** (V1, V2, V3 for video; A1, A2 for audio) with per-track Mute and Solo controls.

### 📝 Motion Graphics & Transforms
*   **Title Overlays:** 4 professional presets (**Lower Third, Center, Subtitle, Kinetic Gold**) with editable fonts, size, and opacity.

## 🛠️ How to Use

1.  **LAUNCH**
    Simply double-click `teamlegacy-editor-pro.html` in any modern browser (Chrome or Edge recommended).
2.  **IMPORT MEDIA**
    Click **"+ Import Media"** or drag video files directly into the **Project Bin**.
3.  **EDITING**
    *   Add clips to the timeline by clicking them in the bin.
    *   Drag clips up/down to switch between Background (V1) and Overlay (V2) tracks.
    *   Use the left panel to adjust Scale and Opacity.
4.  **EXPORT**
    Click **"Export Video"** in the header. The engine will render your timeline frame-by-frame and download the final file automatically.

## ⌨️ Keyboard Shortcuts
| Key | Action |
| :--- | :--- |
| `SPACEBAR` | Play / Pause |
| `S` | Razor Tool (Split Clip) |
| `V` | Selection Tool |
| `T` | Add Title Overlay |
| `J` / `K` / `L` | Playback Speed Control |
| `DELETE` | Remove Selected Clip |
| `Ctrl + Z` / `Y` | Undo / Redo (60-step history) |
| `+/-` | Timeline Zoom |

## 🖥️ System Requirements
*   **Browser:** Chrome or Edge (recommended for optimal rendering performance).
*   **RAM:** 8GB+ recommended for smooth HD editing.
*   **Connection:** An active internet connection is required only to load the initial engine components/fonts.
